# 🎮 Rock Paper Scissors

A simple, interactive Python command-line game.

## ✨ Features
*   **Easy Input:** Use `r`, `p`, or `s` to play.
*   **Emoji Support:** Clean visuals for Rock 🪨, Paper 📃, and Scissors ✂️.
*   **Smart Bot:** Randomized AI opponent.
*   **Error Proof:** Handles invalid inputs without crashing.

## 🚀 How to Run
1. **Clone:**  
   ```bash
   git clone https://github.com/rrbhanderi05/rock-paper-scissors.git
   ```
2. **Enter Directory:**  
   ```bash
   cd rock-paper-scissors
   ```
3. **Play:**  
   ```bash
   python main.py
   ```

## 🛠️ Built With
*   **Python 3**
*   `random` module
*   `while` loops & `if-else` logic
